Function interrupt_thread

hpx::threads::interrupt_thread

Synopsis

// In header: <hpx/runtime/threads/thread_helpers.hpp>


void interrupt_thread(thread_id_type const & id, bool flag, 
                      error_code & ec = throws);

Description

Flag the given thread for interruption.

Note

As long as ec is not pre-initialized to hpx::throws this function doesn't throw but returns the result code using the parameter ec. Otherwise it throws an instance of hpx::exception.

Parameters:

ec

[in,out] this represents the error status on exit, if this is pre-initialized to hpx::throws the function will throw on error instead.

flag

[in] The flag encodes whether the thread should be interrupted (if it is true), or 'uninterrupted' (if it is false).

id

[in] The thread id of the thread which should be interrupted.
